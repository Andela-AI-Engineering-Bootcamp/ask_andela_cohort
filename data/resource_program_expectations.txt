# A3: AI Engineering Bootcamp — Program Expectations
Source: Official program resource page

This self-paced learning experience is complemented by bi-weekly practical workshops designed to support your learning through collaboration while you engage with challenging content.

Mandatory Requirement: You must complete all of the Udemy modules and program deliverables to successfully graduate from the program.

Program Duration: 10 Weeks

---

## Program Overview: Weekly Schedule

Each phase builds on the last. By Week 10, you will have built working AI systems from prompts to production. Behavioral workshops develop the professional skills that matter on client engagements: proactive communication, ownership, adaptive problem-solving, and professional presence.

**Topics covered:** Programme setup, tool access, and orientation. No technical content — this week is entirely administrative. Participants get access to the Udemy course, activate Cursor and GitHub Copilot licences, and attend the kickoff session.

### Onboarding Checklist

- Attend the kickoff session, scheduled for February 18, 2026 at 8:30 PM
- Get access to Course 1 (AI Engineer Core Track) during the kickoff session and ensure you sign in to access the learning materials
- Redeem your licenses for Cursor and GitHub Copilot
- Bookmark the learning program space in the Andela Community (AI Engineering Bootcamp) so you can reference it along the way
- Start learning

## Week 1: LLM Foundations

**Topics covered:** Building your first LLM product, working with top frontier models, building a multi-modal chatbot with Gradio UI, using open-source models via HuggingFace, evaluating models for code generation and business tasks, and an introduction to RAG (Retrieval-Augmented Generation) with vector embeddings. This is the foundational week covering core LLM concepts and tools used throughout the rest of the programme.

Course: AI Engineer Core Track

### Daily Breakdown

- Day 1: Build Your First LLM Product: Exploring Top Models (Day 4–5)
- Day 2: Build a Multi-Modal Chatbot: LLMs, Gradio UI, and Agents. First behavioral competency assignment distributed.
- Day 3: Open-Source Gen AI: Automated Solutions with HuggingFace
- Day 4: Course Material — LLM Showdown: Evaluating Models for Code Gen & Business Tasks. Deep Dive Session (Slides, Recording).
- Day 5: Course Material — Mastering RAG: Build Advanced Solutions with Vector Embeddings. Behavioral competency assignment due today.

## Week 2: LLM Deep Dive

**Topics covered:** Advanced RAG systems using vector embeddings, the path from traditional machine learning to deep learning, fine-tuning frontier models, fine-tuning open-source models to compete with frontier models, and building autonomous multi-agent systems. This is the most technically intensive week of the core track — it covers fine-tuning (including QLoRA concepts), advanced RAG pipelines, and agentic AI.

Course: AI Engineer Core Track

### Daily Breakdown

- Day 2: Deep Dive Session with Ed
- Day 3: From Traditional ML to Deep Learning to Fine-tuning a Frontier Model
- Day 4: Fine-tuned open source model to compete with frontier models. Deep Dive Session with MC.
- Day 5: Build Autonomous Multi-Agent System

## Week 3: AI Strategy and Leadership

**Topics covered:** Transitioning from engineering to strategic AI thinking. Core AI concepts (LLMs, transformers, generative AI) explained for a leadership audience, evaluating frontier vs open-source models, commercial Gen AI strategy, RAG vs fine-tuning decision frameworks, AI agents vs workflows, building AI teams, AI project roadmaps, and delivering ROI. Also includes the first group project (Project 1) where squads build and present a capstone application together.
